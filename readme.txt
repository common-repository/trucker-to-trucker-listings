== Description ==

Used to add your listings from TruckerToTrucker.com to your wordpress website.

== Installation ==

This section describes how to install the plugin, find your personal API Key to only show your listings, and have it display on your website.


1. Upload `truckertotrucker-listings.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3.Log in to your account at http://www.truckertotrucker.com/site/customer_login.cfm
4.Once logged in, go to http://www.truckertotrucker.com/generate-api-key.cfm to get your API key.
5.Copy and save this number.
6.In Wordpress, go to Settings -> TruckerToTrucker Listings.
7.Enter API key and click Submit (the number should be permanently saved now).
8.Add [TruckerToTrucker_Listings] as shortcode to any page that you want to show your listings.
9.The permalink should be changed to Post name. Go to Settings -> Permalinks -> Common Settings -> Post name.

== Frequently Asked Questions ==

= I've added the plugin, but my page says 'File Not Found' =

You need to add your specific API Key to the plugin under Settings -> TruckerToTrucker Listings, and click 'Submit'.

Then reload your page for your listings to show up.

= How do I find my API Key? =

Log into your truckertotrucker.com account and go to http://www.truckertotrucker.com/generate-api-key.cfm.

That number is unique to your listings, and must be added.
